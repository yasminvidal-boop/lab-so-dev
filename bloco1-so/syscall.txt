# PARTE C — Questões de síntese

# 1. A apresentação mostrou que "instruções de I/O são privilegiadas". 
#    Com base no que o strace revelou, explique com suas palavras como um programa
#    de usuário consegue fazer I/O sem executar instruções privilegiadas diretamente.

Resposta 1:
Um programa em modo usuário não executa diretamente instruções privilegiadas de I/O. 
Em vez disso, ele faz uma chamada de sistema (syscall), como write(), que solicita ao 
kernel que execute a operação. O kernel roda em modo privilegiado e pode acessar os 
dispositivos de hardware com segurança.

Resposta 2:
Esse papel é herdado principalmente pelos drivers de dispositivo do sistema operacional,

(Direct Memory Access) permitem transferências de dados entre memória e dispositivos 
com pouca intervenção da CPU, desempenhando função semelhante aos antigos programas de canal.
